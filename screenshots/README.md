# Screenshots

Place screenshots related to detection scenarios, SIEM dashboards, or alert evidence in this folder.

## Naming Convention

```
<scenario>-<description>-<YYYYMMDD>.png
```

**Examples**
- `scenario1-powershell-alert-20240101.png`
- `scenario2-bruteforce-dashboard-20240101.png`
- `scenario3-persistence-registry-20240101.png`
